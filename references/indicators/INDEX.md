# 技术指标模块索引

> **指标是"辅助确认工具"，不是"独立入场信号"。先有结构判断和形态信号，再用指标确认。**

| 文件 | 指标 | 适用场景 |
|------|------|---------|
| rsi.md | RSI | 超买超卖、背离分析、趋势确认 |
| macd.md | MACD | 背离分析、趋势动量 |
| smart-money-concepts.md | SMC | 供需区、FVG、流动性抓取 |

**触发关键词路由**：
- RSI、超买、超卖、RSI背离 → `rsi.md`
- MACD、金叉、死叉、MACD背离、柱状图 → `macd.md`
- 聪明钱、SMC、供需区、FVG、流动性、BOS/CHoCH → `smart-money-concepts.md`
