# 聪明钱概念 (Smart Money Concepts / SMC)

## 基本信息

| 属性 | 值 |
|------|---|
| 类型 | 结构分析框架（非传统指标） |
| 核心理念 | 跟随机构/大资金的行为，而非散户 |
| 适用市场 | 所有市场（股票、期货、加密货币） |

---

## 核心概念

### 聪明钱 vs 散户

| 类型 | 特征 |
|------|------|
| **聪明钱**（Smart Money） | 银行、对冲基金、机构 — 资金量大，能影响市场走向 |
| **散户**（Retail Traders） | 个人交易者 — 资金量小，通常被聪明钱"猎杀" |

---

## BOS & CHoCH（与市场结构共通）

### BOS（Break of Structure / 结构突破）

- **顺趋势方向**突破前高/前低 → 趋势延续信号

### CHoCH（Change of Character / 性格转变）

- **逆趋势方向**突破 → 趋势可能反转的预警

### 内部与外部结构（多时间框架）

| 类型 | 说明 |
|------|------|
| **外部 BOS/CHoCH** | 发生在高时间框架（如日线） |
| **内部 BOS/CHoCH** | 发生在低时间框架（如1小时线） |

---

## 供需区 (Supply & Demand Zones)

> **核心逻辑**：价格回到历史供需失衡区域时，该失衡往往会再次出现

### 识别需求区（Demand Zone）— 三步法

1. **找动量蜡烛**：实体至少是前几根的 **2 倍**（3~5倍更佳），方向为大阳线
2. **找前一根蜡烛**：即动量蜡烛的前一根
3. **画区域**：从前一根蜡烛的最低点到最高点画矩形

### 识别供给区（Supply Zone）

与需求区镜像：找大阴线动量蜡烛 → 前一根蜡烛 → 画区域

### 供需区交易流程

```
1. 确认趋势方向（上升趋势找需求区做多）
2. 等待 BOS，确认强低点
3. 在强低点附近画需求区
4. 等待价格回落至需求区
5. 在需求区内等待看涨信号（如锤子线、吞没形态）
6. 入场 → 止损 → 目标位
```

### 入场/止损/目标

| 要素 | 方法 |
|------|------|
| **入场** | K线形态确认后的收盘价 |
| **止损** | 需求区/供给区**外侧**（含K线形态低/高点） |
| **目标** | 前期高/低点下方 或 固定风险回报（2:1 / 3:1） |

---

## FVG（Fair Value Gap / 公允价值缺口）

### 看涨 FVG

```
FVG 区域 = 蜡烛①最高点 到 蜡烛③最低点 之间的空白区域
```

需要连续 3 根蜡烛，中间蜡烛（②）的实体很大，使①和③之间出现无重叠区域。

### 看跌 FVG

```
FVG 区域 = 蜡烛①最低点 到 蜡烛③最高点 之间的空白区域
```

### FVG 交易方法

1. 识别发生在 **BOS 行情中** 的 FVG（优先选择）
2. 等待价格**回测 FVG 区域**
3. 在 FVG 内等待看涨/看跌信号
4. 确认后入场

## 流动性 (Liquidity)

### 流动性抓取（Liquidity Grab）

> 聪明钱先打掉散户的止损（抓取流动性），然后才真正推动价格

### 流动性抓取 vs CHoCH 的区别

| 判断依据 | 流动性抓取 | 真实 CHoCH |
|----------|-----------|-----------|
| K线特征 | **影线**刺穿后快速收回 | **实体收盘价**跌破/突破 |
| 典型蜡烛 | 锤子线（长影线快速收回） | 大阴/阳线实体收破关键位 |

> 📌 **影线穿越 = 流动性抓取；实体收破 = 真实突破**

### 交易流动性抓取

```
1. 等待价格突破关键支撑/阻力
2. 观察是否快速收回（影线 vs 实体）
3. 如果收回 → 流动性抓取 → 反向入场
4. 止损：抓取的极端点外侧
5. 目标：前期结构位
```

## 完整 SMC 交易流程

```
第一步：确认高时间框架趋势方向
第二步：找到 BOS（确认趋势方向有效）
第三步：标记供需区（找动量蜡烛+前一根蜡烛）
第四步：等待价格回到供需区
第五步：在供需区内等待 K线形态确认
第六步：执行交易
  ├── 入场：信号K线收盘价
  ├── 止损：区域下方
  └── 目标：前期高点下方 或 固定风险回报（2:1 / 3:1）
```

## 速查表

| 概念 | 缩写 | 核心含义 |
|------|------|---------|
| 结构突破 | BOS | 顺势突破，趋势延续 |
| 性格转变 | CHoCH | 逆势突破，趋势可能反转 |
| 需求区 | Demand Zone | 动量蜡烛起点，强支撑 |
| 供给区 | Supply Zone | 动量蜡烛起点，强阻力 |
| 公允价值缺口 | FVG | 快速移动留下的"空白"，价格倾向回填 |
| 流动性抓取 | Liquidity Grab | 影线刺穿后收回 = 假突破 |

### 五大核心原则

1. **跟随聪明钱**：分析机构行为而非散户情绪
2. **供需区优于传统支撑阻力**：基于动量蜡烛识别
3. **BOS 确认趋势方向**：不要在没有 BOS 的情况下交易
4. **影线 vs 实体**：区分流动性抓取和真实突破
5. **多时间框架**：高时间框架定方向，低时间框架找精确入场点
