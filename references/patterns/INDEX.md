## K线形态决策树

```
用户描述 K线 →
│
├── 单根 K线
│   ├── 长下影线 + 小实体 + 无/短上影线
│   │   ├── 之前下跌？ → 🟢 锤子线 (Hammer)
│   │   └── 之前上涨？ → 🔴 上吊线 (Hanging Man)
│   │
│   ├── 长上影线 + 小实体 + 无/短下影线
│   │   ├── 之前上涨？ → 🔴 射击之星 (Shooting Star)
│   │   └── 之前下跌？ → 🟢 倒锤子 (Inverted Hammer)
│   │
│   ├── 开盘 ≈ 收盘（极小或无实体）
│   │   ├── 长下影线 + 无上影线？ → 蜻蜓十字星
│   │   ├── 长上影线 + 无下影线？ → 墓碑十字星
│   │   └── 均等长影线？ → 长脚十字星
│   │
│   ├── 实体 ≥ 前3根的 2 倍？ → 动量蜡烛
│   └── 小实体 + 上下影线均 > 实体？ → 纺锤顶（犹豫信号）
│
├── 2 根 K线
│   ├── 第2根实体完全包住第1根实体？
│   │   ├── 下跌后 + 阳包阴？ → 🟢 看涨吞没
│   │   └── 上涨后 + 阴包阳？ → 🔴 看跌吞没
│   └── 第2根收于第1根实体 50% 以上/以下？
│       ├── 下跌后 + 阳线收 > 50%？ → 🟢 刺穿线
│       └── 上涨后 + 阴线收 < 50%？ → 🔴 乌云盖顶
│
├── 3 根 K线
│   └── 阴 → 小 → 阳（下跌后）？ → 🟢 晨星
│       阳 → 小 → 阴（上涨后）？ → 🔴 黄昏星
│
└── 5 根 K线
    ├── 阳 → 3小阴 → 阳？ → 🟢 上升三法
    └── 阴 → 3小阳 → 阴？ → 🔴 下降三法
```

## 图表形态决策树

```
Step 1: 确定前置趋势（上升/下降）
Step 2: 匹配结构特征
    两个等高高点 + 中间低谷？ → 双顶
    两个等低低点 + 中间高峰？ → 双底
    等高高点 + 抬升低点？ → 上升三角形
    降低高点 + 等低低点？ → 下降三角形
    降低高点 + 降低低点 + 收窄？ → 下降楔形
    抬升高点 + 抬升低点 + 收窄？ → 上升楔形
    强冲量 + 小幅回调Channel？ → 旗形
    左肩 + 头 + 右肩？ → 头肩形态
    U形回调 + 小回调柄？ → 杯柄形态
    降低高点 + 抬升低点（对称）？ → ⚠️ 对称三角形（避免交易）
    抬升高点 + 降低低点（扩散）？ → ⚠️ 宽幅震荡（避免交易）
```
