## AI 形态分析黄金法则

> **在识别任何形态之前，AI 必须遵循以下原则：**

1. **位置 > 形态**：先判断趋势方向和关键位（支撑/阻力），再看形态。同一形态在不同位置含义完全不同
2. **多重确认**：单一形态不足以入场，至少需要 2 个独立确认因素
3. **容错识别**：真实市场形态不完美，核心条件满足即可，不要求教科书级完美
4. **混淆检查**：对于外形相似的形态（锤子/上吊、射击之星/倒锤子），**必须先确认前置趋势**
5. **强度分级**：给出信号强度评级（弱/中/强），而非简单的"是/否"
6. **始终附带风控**：每次形态分析必须包含入场点、止损位、目标位

## K线形态识别决策树

AI 根据用户描述的 K线特征，按以下路径定位形态：

```
用户描述 K线 →
│
├── 单根 K线（简单形态）
│   ├── 长下影线 + 小实体 + 无/短上影线
│   │   ├── 之前下跌？ → 🟢 锤子线 (Hammer)
│   │   └── 之前上涨？ → 🔴 上吊线 (Hanging Man)
│   │
│   ├── 长上影线 + 小实体 + 无/短下影线
│   │   ├── 之前上涨？ → 🔴 射击之星 (Shooting Star)
│   │   └── 之前下跌？ → 🟢 倒锤子 (Inverted Hammer)
│   │
│   ├── 开盘 ≈ 收盘（极小或无实体）
│   │   ├── 长下影线 + 无上影线？ → 蜻蜓十字星（看位置定方向）
│   │   ├── 长上影线 + 无下影线？ → 墓碑十字星（看位置定方向）
│   │   ├── 均等长影线？ → 长脚十字星
│   │   └── 无影线？ → 四价十字星（不宜交易）
│   │
│   ├── 实体 ≥ 前3根的 2 倍？ → 动量蜡烛（顺趋势 = 延续）
│   │
│   └── 小实体 + 上下影线均 > 实体？ → 纺锤顶（犹豫信号）
│
├── 2 根 K线
│   ├── 第2根实体完全包住第1根实体？
│   │   ├── 下跌后 + 阳包阴？ → 🟢 看涨吞没
│   │   └── 上涨后 + 阴包阳？ → 🔴 看跌吞没
│   │
│   └── 第2根收于第1根实体 50% 以上/以下？
│       ├── 下跌后 + 阳线收 > 50%？ → 🟢 刺穿线
│       └── 上涨后 + 阴线收 < 50%？ → 🔴 乌云盖顶
│
├── 3 根 K线
│   └── 大K → 小K → 大K（方向反转）
│       ├── 阴 → 小 → 阳（下跌后）？ → 🟢 晨星
│       └── 阳 → 小 → 阴（上涨后）？ → 🔴 黄昏星
│
├── 5 根 K线
│   └── 大K → 3小K回调 → 大K突破
│       ├── 阳 → 3小阴 → 阳？ → 🟢 上升三法
│       └── 阴 → 3小阳 → 阴？ → 🔴 下降三法
│
└── 多K线结构（较长时间）
    ├── 强冲量波 + 小幅回调 + 突破？ → 旗形（查 candlestick-patterns.md）
    └── 更大结构？ → 查 chart-patterns.md
```

## 图表形态识别路由

```
用户描述图表结构 →
│
├── Step 1: 确定前置趋势
│   ├── 上升趋势？ → 关注看跌反转形态 或 看涨延续形态
│   └── 下降趋势？ → 关注看涨反转形态 或 看跌延续形态
│
├── Step 2: 匹配结构特征
│   ├── 两个等高高点 + 中间低谷？ → 双顶
│   ├── 两个等低低点 + 中间高峰？ → 双底
│   ├── 等高高点 + 抬升低点？ → 上升三角形
│   ├── 降低高点 + 等低低点？ → 下降三角形
│   ├── 降低高点 + 降低低点 + 收窄？ → 下降楔形
│   ├── 抬升高点 + 抬升低点 + 收窄？ → 上升楔形
│   ├── 强冲量 + 小幅回调Channel？ → 旗形
│   ├── 左肩 + 头(更高/低) + 右肩？ → 头肩形态
│   ├── U形回调 + 小回调柄？ → 杯柄形态
│   ├── 降低高点 + 抬升低点(对称)？ → ⚠️ 对称三角形（避免交易）
│   └── 抬升高点 + 降低低点(扩散)？ → ⚠️ 宽幅震荡（避免交易）
│
└── Step 3: 查 chart-patterns.md 中对应形态章节的识别检查清单获取详细验证条件
```

## AI 加载指引

- 用户描述 **单根/少数 K线** → 加载 `candlestick-patterns.md`
- 用户描述 **多K线大结构** → 加载 `chart-patterns.md`
- 用户只说"帮我看看形态" → 先用本文件决策树初判，再按需加载具体文件
- **形态确认后**，建议联动加载 `references/core/support-resistance.md` 确认关键位
