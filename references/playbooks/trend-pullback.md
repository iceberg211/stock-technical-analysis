# 顺势回调 (Trend Pullback)

> **在健康趋势中，等回调到位、等触发出现、顺势入场。**

---

## 前提条件（缺一不可）

| 条件 | 标准 |
|------|------|
| 趋势清晰 | HH+HL（做多）或 LL+LH（做空），至少 2 个同方向 BOS |
| 推动浪健康 | Impulse 实体大、方向一致、速度快 |
| 没有 CHoCH | 最近不能出现反向 CHoCH |
| HTF 同向 | 高周期趋势方向与本级别一致 |

---

## 回调质量评估

### 健康回调（✅ 可以等触发）— 核心条件全部满足

| 特征 | 说明 |
|------|------|
| K线特征 | 小实体、影线多、颜色混杂 |
| 结构保持 | 未破坏前一个 Major Swing（做多时未跌破前 HL） |
| 回调速度 | 比推动浪慢 |
| 无反向动量 | 未出现与推动浪等量级的反向K线 |
| 增强（可见时）| Fib 0.382–0.618 区间内；成交量递减 |

### 不健康回调（❌ 不做）— 任一出现即放弃

| 特征 | 说明 |
|------|------|
| 回调过猛 | 出现反向动量K线（实体 ≥ 推动浪K线） |
| CHoCH 出现 | 回调破坏了趋势内部结构 |
| 跌破关键位 | 破坏了趋势的 Strong Level |
| 增强否决（可见时）| 超过 Fib 0.786；放量下杀 |

---

## 触发信号

### 做多回调触发

| 触发类型 | 具体条件 | 可靠性 |
|---------|---------|-------|
| **反转K线** | Fib 0.382–0.618 区域出现锤子线/吞没/启明星 | ⭐⭐⭐ |
| **低级别 BOS** | 低时间框架反向 CHoCH + 同向 BOS | ⭐⭐⭐⭐ |
| **支撑拒绝** | 触碰前 HL 区域 + 长下影线 + 缩量 | ⭐⭐⭐ |
| 指标确认 | RSI 从超卖区回升 / MACD 柱缩小并反转 | ⭐⭐（辅助） |

### 做空回调触发（镜像）

| 触发类型 | 具体条件 |
|---------|---------|
| **反转K线** | Fib 0.382–0.618 区域出现射击之星/吞没/黄昏星 |
| **低级别 BOS** | 低时间框架反向 CHoCH + 空头 BOS |
| **阻力拒绝** | 触碰前 LH 区域 + 长上影线 |

> **没有触发就不入场。**

---

## 入场规则

| 类型 | 时机 | 风险 |
|------|------|------|
| **激进入场** | 反转K线收盘后，下一根开盘入场 | 可能被进一步回调扫止损 |
| **保守入场** | 等低级别同向 BOS 确认后入场 | 入场价劣势，但止损更紧 |

---

## 止损规则

| 方向 | 止损位置 |
|------|---------|
| 做多 | 回调最低点下方 + buffer（ATR × 0.3–0.5；无ATR时取回调段最大影线长度或止损位的 0.5–1%） |
| 做空 | 回调最高点上方 + buffer |

> 止损距离过大导致 R:R < 1.5:1 → **放弃此笔交易**

---

## 目标设定

| 目标 | 位置 | 操作 |
|------|------|------|
| **T1** | 前高/前低（趋势方向最近 Swing 极值） | 到达后减仓 50%，止损移至成本价 |
| **T2** | 等距运动或下一个关键阻力/支撑 | 剩余仓位跟踪止损 |
| T3（可选）| 更高级别阻力/支撑 | 仅强趋势中考虑 |

---

## 不做清单

| 条件 | 原因 |
|------|------|
| 趋势已走 3 波以上 | 趋势末端，胜率骤降 |
| 回调超过 0.786 | 已不是回调，是可能反转 |
| HTF 方向冲突 | 逆大势做小势 |
| 重大事件即将公布 | 技术分析可能失效 |
| 同品种已有同向持仓 | 避免重复暴露 |
| 前一次同类 setup 刚被止损 | 等市场重新证明结构 |
