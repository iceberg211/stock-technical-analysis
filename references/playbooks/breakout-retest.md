# 突破回踩 (Breakout Retest)

> **突破不是入场信号，回踩确认才是。**

---

## 有效突破的定义

**核心条件（必须全部满足）：**

| 条件 | 标准 |
|------|------|
| 动量K线 | 突破K线实体 ≥ 前 3 根的 2 倍 |
| 收盘越过 | 实体收盘在关键位**之外**，不仅是影线穿过 |
| 关键水平位 | 被突破的是真正的关键支撑/阻力 |
| 增强（可见时）| 突破时成交量明显放大 |

**假突破识别（任一出现 → 突破无效）：**

| 信号 | 说明 |
|------|------|
| 仅影线穿过 | 实体收回，没有真正站上/跌破 |
| 无量突破 | 成交量没有明显放大 |
| 收盘失败 | 突破K线收盘回到关键位内侧 |
| 立即反向 | 下一根K线出现反向动量K线 |
| 连续小K线试探 | 多根小K线反复试探，无一有力度 |

---

## 回踩等待规则

**回踩接单条件（核心必须满足）：**

| 条件 | 标准 |
|------|------|
| 价格回到突破位附近 | 回踩到原关键位附近 |
| 回踩K线温和 | 小实体、影线多，不是动量K线反砸 |
| 出现拒绝PA | 在回踩位出现反转K线/长影线/低级别反转结构 |
| 增强（可见时）| 回踩缩量；回踩到原关键位 ± ATR × 0.5 以内 |

**回踩失败判定：**

| 信号 | 行动 |
|------|------|
| 回踩跌破突破起点 | ❌ 方案彻底失效 |
| 回踩带动量K线反穿 | ❌ 突破已被否定，可能是假突破 |
| 回踩后横盘 > 20 根K线 | ⚠️ 信号衰减，降级或放弃 |

---

## 不回踩例外（默认不追，必须同时满足4条才可）

| 条件 | 标准 |
|------|------|
| 极强趋势 | HTF 趋势清晰 + 当前突破是 BOS 延续 |
| 连续动量 | 突破后连续 2 根以上同向动量K线 |
| 小级别回调 | 低时间框架浅回调（Fib 0.236–0.382） |
| 仓位缩减 | 仓位降至正常的 50% |

---

## 入场规则

| 类型 | 时机 | 止损 |
|------|------|------|
| **标准入场** | 回踩到位 + 拒绝PA → 拒绝K线收盘后入场 | 回踩最低/最高点外 + buffer |
| **激进入场** | 回踩刚触碰突破位即入场 | 突破起点之外（仓位必须缩减） |
| **不回踩例外** | 低时间框架浅回调后 | 突破K线实体中点 |

> buffer：首选 ATR × 0.3–0.5；无ATR时取回踩段最大影线长度，或止损位的 0.5–1%

---

## 目标设定

| 目标 | 位置 | 操作 |
|------|------|------|
| **T1** | 突破幅度等距投射（Measured Move）| 减仓 50%，止损移至成本价 |
| **T2** | 下一个关键阻力/支撑 | 剩余仓位跟踪止损 |

等距投射：做多 T1 = 突破位 + (突破位 - 前Swing Low)；做空 T1 = 突破位 - (前Swing High - 突破位)

---

## 不做清单

| 条件 | 原因 |
|------|------|
| 突破不满足有效性标准 | 可能是假突破 |
| 回踩跌破/突破起点 | 翻转失败 |
| HTF 方向与突破方向冲突 | 失败率高 |
| 区间被来回穿多次 | 是噪音，不是突破 |
| 重大事件即将公布 | 技术信号可能被否定 |
| 同时匹配 trend-pullback | 优先按 trend-pullback 处理 |
