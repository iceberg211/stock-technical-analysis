# 假突破反转 (False Breakout Reversal)

> **市场最喜欢的把戏：先骗你突破，再反向收割。假突破 = 最好的反向入场机会之一。**
> 这个 Playbook 把"被骗"变成武器——当突破失败时，被困的一方急于出逃，为反向提供燃料。

---

## 一、假突破的定义

### 必须满足的标准

| 条件 | 标准 |
|------|------|
| 关键位被穿越 | 价格越过了明确的支撑/阻力/区间边界 |
| 快速收回 | 1~3 根 K 线内，价格实体收回关键位内侧 |
| 影线特征 | 突破 K 线留下明显的长影线（影线 >= 实体 1.5 倍） |
| 未形成后续动量 | 突破方向没有出现连续的同向动量 K 线 |

### 假突破 vs 真突破的区分

| 判断依据 | 假突破 | 真突破 |
|----------|--------|--------|
| K 线收盘 | 收回关键位内侧 | 实体收盘在关键位外侧 |
| 影线 | 留长影线 | 影线短或无 |
| 后续 K 线 | 反向动量出现 | 同向延续 |
| 成交量（可见时） | 突破时放量但收回也放量（争夺） | 突破放量 + 后续缩量回踩 |

---

## 二、常见假突破场景

| 场景 | 说明 | 频率 |
|------|------|------|
| **区间假突破** | 震荡区间上/下沿被短暂突破后收回 | 最常见 |
| **趋势假突破** | 趋势中新高/新低后快速反转（如双顶假突破） | 常见 |
| **关键位假突破** | 重要支撑/阻力被刺穿后收回 | 常见 |
| **形态假突破** | 三角形/旗形提前突破后失败 | 较少但利润大 |

---

## 三、触发条件（假突破确认后等待以下出现）

### 做多（下方假突破后反向做多）

| 触发类型 | 具体条件 | 可靠性 |
|---------|---------|-------|
| **Pin Bar 收回** | 长下影线刺穿支撑后实体收回支撑上方 | 三星 |
| **吞没反转** | 假突破 K 线后出现看涨吞没 | 三星 |
| **快速 V 型收回** | 1~2 根 K 线完成刺穿 + 收回 + 反向动量 | 四星 |
| **低级别结构反转** | 低时间框架出现 CHoCH + 反向 BOS | 四星 |

### 做空（上方假突破后反向做空）

与做多镜像：长上影线收回阻力下方、看跌吞没、快速倒 V 收回、低级别空头结构。

> 关键：假突破确认 = 价格**实体收盘**回到关键位内侧。仅影线回收不算确认，需等收盘。

---

## 四、入场规则

### 标准入场（推荐）

- **时机**：假突破确认 K 线收盘后，下一根开盘入场
- **方向**：与突破方向相反（假突破上方 → 做空，假突破下方 → 做多）
- **条件**：确认 K 线必须实体收回关键位内侧

### 激进入场

- **时机**：假突破 K 线本身就是 Pin Bar 且已形成长影线，不等下一根
- **条件**：影线 >= 实体 2 倍 + 关键位明确
- **风险**：可能还没完全收回，仓位须缩减至 50%

---

## 五、止损规则

| 方向 | 止损位置 | 说明 |
|------|---------|------|
| 下方假突破做多 | 假突破最低点下方 + buffer | 如果再次跌破 = 不是假突破 |
| 上方假突破做空 | 假突破最高点上方 + buffer | 如果再次突破 = 不是假突破 |

> buffer：首选 ATR x 0.3~0.5；无 ATR 时取假突破 K 线影线长度的 20%，或用止损位的 0.5~1% 作为固定偏移。

### 止损优势

假突破的止损通常很紧（因为极值点就在附近），这使得 R:R 天然较好。

---

## 六、目标设定

| 目标 | 位置 | 操作 |
|------|------|------|
| **T1** | 区间/结构的对侧关键位（区间反转时 = 对面沿；趋势中 = 前一个 Swing） | 到达后减仓 50%，止损移至成本价 |
| **T2** | 更远的结构位或等距运动 | 剩余仓位跟踪止损 |

> 假突破的目标取决于**假突破发生的上下文**：区间内假突破目标有限（对侧沿）；趋势反转假突破目标更大。

---

## 七、不做清单（Disqualifiers）

| 条件 | 原因 |
|------|------|
| 突破后已走出 3 根以上同向动量 K 线 | 不是假突破，是真突破后的回调 |
| 突破方向与 HTF 趋势一致 | 高概率是真突破的正常回踩 |
| 突破发生在低流动性时段（如亚盘突破欧美盘关键位） | 假信号率高但不可靠 |
| 关键位本身不够清晰（仅触碰 1 次的"支撑/阻力"） | 不是真正的关键位，假突破逻辑不成立 |
| 重大事件即将公布 | 事件可能使价格重新突破 |

## 完整流程图

```
关键位被突破？
  |
  +-- 否 -> 不是假突破场景
  |
  +-- 是 -> 价格 1~3 根 K 线内实体收回关键位内侧？
          |
          +-- 否 -> 可能是真突破，转 breakout-retest.md
          |
          +-- 是 -> 出现反向触发信号？（Pin Bar / 吞没 / 低级别反转）
                  |
                  +-- 否 -> 等待触发或放弃
                  |
                  +-- 是 -> HTF 方向与反向交易一致或中性？
                          |
                          +-- 否 -> 降仓 50% 或放弃
                          |
                          +-- 是 -> 通过 pre-trade-checklist？
                                  |
                                  +-- 否 -> 被过滤掉
                                  |
                                  +-- 是 -> 入场
                                          -> 止损：假突破极值外
                                          -> T1：对侧关键位
```

## AI 使用指引

- 用户问"这是假突破吗" -> 按 4 条假突破标准逐项检查
- 看到突破后快速收回 -> 主动提示"可能是假突破，观察是否实体收回"
- 假突破确认后 -> 引导用户考虑反向入场，而非追原方向
- 区间假突破 -> 可与 range-reversal 配合使用（假突破提供更好的入场点）
- 给出方案时必须明确"这是假突破反向单"，标注止损在假突破极值外
