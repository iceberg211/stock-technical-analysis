# 区间边缘反转 (Range Reversal)

> **震荡机会只在边缘，区间中间是禁区。**

---

## 成熟区间的定义

| 条件 | 标准 |
|------|------|
| 触碰次数 | 上沿和下沿各至少被触碰 **2 次**（合计 ≥ 4 次） |
| 高低点走平 | Swing High 大致走平、Swing Low 大致走平 |
| 时间长度 | 区间存在至少 **20 根 K 线** |
| 边界清晰 | 上下沿可画出水平线/窄区域 |

**不是区间（❌ 不套用此 Playbook）：**

| 情况 | 说明 |
|------|------|
| 触碰不足 | 少于 2 次上下沿触碰 |
| 区间内有明显 BOS | 那是趋势中的盘整 |
| 上下沿越来越窄 | 三角形收敛，参考 `patterns/chart-patterns.md` |
| 波动剧烈无序 | 混乱市，不做 |

---

## 可做区域 vs 禁区

- **上沿 20% 区域** ← ✅ 可做空（如有拒绝信号）
- **中间 60% 区域** ← ❌ 禁区，无结构优势
- **下沿 20% 区域** ← ✅ 可做多（如有拒绝信号）

计算：上沿区域 = 上沿往下 (区间高度 × 20%)；下沿区域 = 下沿往上 (区间高度 × 20%)

---

## 触发条件（在边缘 20% 区域内出现才有效）

### 下沿做多触发

| 触发类型 | 具体条件 | 可靠性 |
|---------|---------|-------|
| **长下影线拒绝** | 长下影线（影线 ≥ 实体 2 倍） | ⭐⭐⭐ |
| **反转K线形态** | 锤子线/看涨吞没/启明星 | ⭐⭐⭐ |
| **多次触碰不破** | 第 3/4 次触碰下沿无法跌破 | ⭐⭐⭐⭐ |
| **低级别反转结构** | 低时间框架 CHoCH + 多头 BOS | ⭐⭐⭐⭐ |
| RSI 超卖反弹 | RSI < 30 且开始回升 | ⭐⭐（辅助） |

### 上沿做空触发（镜像）

| 触发类型 | 具体条件 |
|---------|---------|
| **长上影线拒绝** | 在上沿区域出现长上影线 |
| **反转K线形态** | 射击之星/看跌吞没/黄昏星 |
| **多次触碰不破** | 第 3/4 次触碰上沿无法突破 |
| **低级别反转结构** | 低时间框架 CHoCH + 空头 BOS |
| RSI 超买回落 | RSI > 70 且开始回落 |

---

## 入场规则

| 类型 | 时机 |
|------|------|
| **标准入场** | 边缘触发K线收盘后，下一根开盘入场 |
| **保守入场** | 等低级别反转 BOS 后入场（确认度更高，止损更紧） |

---

## 止损规则

| 方向 | 止损位置 |
|------|---------|
| 下沿做多 | 区间下沿之外 + buffer |
| 上沿做空 | 区间上沿之外 + buffer |

> buffer：首选 ATR × 0.5；无ATR时取区间内最大影线长度，或止损位的 0.5–1%

---

## 目标设定

| 目标 | 位置 | 操作 |
|------|------|------|
| **T1** | 区间中轴 = (上沿 + 下沿) ÷ 2 | 减仓 50%，止损移至成本价 |
| **T2** | 区间对侧 | 接近对侧时考虑平仓 |

---

## 区间失效 → 转突破

| 信号 | 行动 |
|------|------|
| 动量K线突破上/下沿 | 切换到 `breakout-retest.md` |
| 突破 + 回踩确认翻转 | 执行突破回踩流程 |
| 区间内出现 BOS | 重新评估市场状态 |

---

## 不做清单

| 条件 | 原因 |
|------|------|
| 区间不成熟（< 4 次触碰） | 可能不是真正的区间 |
| 价格在区间中间 | 无结构优势 |
| 没有拒绝信号就入场 | 在边缘 ≠ 必然反转 |
| HTF 明确方向 + 在趋势方向的沿 | 区间可能被突破 |
| 区间越来越窄 | 是收敛三角形 |
| 重大事件即将公布 | 区间依赖稳定环境 |

> HTF偏多 → 下沿做多信心更高，上沿做空信心降级；HTF偏空 → 反之
