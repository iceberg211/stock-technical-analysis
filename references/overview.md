# 技术分析总纲 (Technical Analysis Overview)

> 技术分析的核心假设：**价格反映一切，历史会重演，趋势会延续**

---

## 分析层级（从宏观到微观）

| 层级 | 模块 | 核心问题 | 优先级 |
|------|------|---------|--------|
| **1. 方向判断** | 市场结构 | 该做多还是做空？ | 🔴 最高 |
| **2. 关键位标记** | 支撑与阻力 | 在哪里做？ | 🔴 最高 |
| **3. 入场时机** | 价格行为 + K线形态 | 什么时候做？ | 🟡 高 |
| **4. 大结构确认** | 图表形态 | 有没有更大的结构支持？ | 🟡 高 |
| **5. 辅助确认** | RSI / MACD / SMC | 指标是否确认？ | 🟢 中 |
| **6. Setup 匹配** | Playbooks | 这是哪一类交易？ | 🔴 最高 |
| **7. 入场检查** | Pre-trade Checklist | 能不能做这笔？ | 🔴 最高 |
| **8. 仓位/风控** | Position Sizing | 该下多大？ | 🔴 最高 |

> ⚠️ **1-5 是「会分析」，6-8 是「会执行」。两者缺一不可。**

---

## 核心原则

| 原则 | 内容 |
|------|------|
| **位置 > 形态** | 同一形态在不同位置含义不同 |
| **结构 > 信号** | 先看宏观结构，再看微观信号 |
| **多重确认** | 至少 2 个独立因素确认才入场 |
| **概率思维** | 所有信号都是概率，没有确定性 |
| **风控第一** | 好交易 = 结构 + 行为 + 风控 |
| **容错识别** | 真实市场不完美，核心条件满足即可 |
| **趋势为友** | 顺势交易胜率远高于逆势 |
| **会不做** | 最成熟的能力是识别不该交易的时刻 |
