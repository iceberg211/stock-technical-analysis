# 仓位计算与风控规则 (Position Sizing)

> **仓位大小由"你愿意亏多少"加"止损在哪"反推，不是感觉出来的。**

---

## 核心公式

```
仓位 = 风险金额 ÷ 止损距离

其中：
  风险金额 = 总资金 × 单笔风险百分比
  止损距离 = |入场价 - 止损价|
```

---

## 单笔风险上限

| 参数 | 标准值 |
|------|-------|
| **正常单笔风险** | 总资金的 **1–2%** |
| **高信心交易** | 最多 **2%** |
| **降仓交易** | **0.5–1%** |
| **绝对上限** | **3%** |

---

## 降仓规则

| 情况 | 仓位调整 |
|------|---------|
| HTF 方向不明/震荡 | 正常 × 50% |
| 逆势交易（允许时） | 正常 × 50% |
| 事件日交易 | 正常 × 50% |
| 连亏后（见下） | 按连亏规则降 |
| R:R 在 1.5:1–2:1 之间 | 正常 × 75% |
| 激进入场（不回踩例外） | 正常 × 50% |

---

## 连续亏损处理

| 连亏笔数 | 行动 |
|---------|------|
| 连亏 **2 笔** | ⚠️ 自查：setup 问题还是执行问题？ |
| 连亏 **3 笔** | 🔴 仓位自动降至正常的 **50%** |
| 连亏 **5 笔** | 🔴🔴 暂停交易一天（或一个时段） |
| 连亏 **7 笔** | 🔴🔴🔴 暂停 + 完整复盘 + 请求外部反馈 |

**恢复规则**：暂停后恢复 → 前 3 笔用 50% 仓位，3 笔盈利后恢复正常；连亏后首笔盈利不立即满仓，需连续 2 笔盈利

---

## 日/周最大亏损（熔断）

| 级别 | 上限 | 触发行动 |
|------|------|---------|
| 日最大亏损 | 总资金 **3–5%** | 当日停止交易 |
| 周最大亏损 | 总资金 **8–10%** | 本周停止交易 |

---

## 相关性风控

| 规则 | 说明 |
|------|------|
| 同一品种 | 最多 **1 笔** 持仓 |
| 高相关品种同方向 | 最多 **2 笔**（BTC/ETH、黄金/白银、纳指/标普等） |
| 总暴露上限 | 所有持仓总风险不超过总资金 **5%** |

---

## 杠杆使用规则

| 规则 | 说明 |
|------|------|
| 杠杆不改变仓位逻辑 | 名义仓位（股数/合约数）按公式计算，不因杠杆放大 |
| 保证金要求 | 保证金比例保持 > 2 倍维持保证金 |
| 高杠杆警告 | 杠杆 > 10× → 止损必须是硬止损，不能扛 |
