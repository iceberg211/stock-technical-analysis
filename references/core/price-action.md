# 价格行为 (Price Action)

> **结构决定「在哪看」，价格行为决定「怎么做」。**

---

## K线底层逻辑

### 实体（Body）
- **大实体**：一方明显占优 → 突破 / 延续
- **小实体**：犹豫、拉扯 → 拒绝 / 吸收 / 犹豫

### 影线（Wick）
- 长下影线：下方被强力拒绝（买压）
- 长上影线：上方被强力拒绝（卖压）

**核心公式**：
> 大实体 + 小影线 → 突破 / 延续  
> 小实体 + 长影线 → 拒绝 / 吸收 / 犹豫

---

## Momentum Candle（动量K线）

**定义**：实体 ≥ 前 3–5 根K线的 **2 倍**（理想 3–4 倍），影线很短，收盘靠近极值

**序列性**：后续同等大小或更大 → 动量延续；明显缩小 → 动量重置

---

## 支撑阻力中的 Price Action

| 信号 | 特征 |
|------|------|
| **强支撑** | 长影线拒绝 + 明确反转形态 + 快速远离 + 放量 |
| **支撑失效** | 低点不断降低 + 小K堆积 + 多次测试 + 放量却不反弹 |

---

## 突破（Breakout）

**有效突破必须同时满足**：① Momentum Candle　② 明确收盘越过　③ 放量　④ 关键、明显水平位

| 方式 | 说明 |
|------|------|
| 直接追突破 | 需要动量确认 |
| 等回踩确认 | 原阻力 → 支撑 |

**假突破特征**：只有影线穿越（无实体收过）+ 无量 + 收盘失败 + 立刻反向K线

## Volume 在 PA 中的角色

| 量价组合 | 信号 |
|---------|------|
| 高量大K | ✅ 可信 |
| 低量大K | ⚠️ 可疑 |
| 放量反转 | ✅ 可信 |
| 缩量突破 | ⚠️ 警惕 |
| 量价背离 | ⚠️ 趋势衰竭 |
