# 价格行为 (Price Action)

> **结构决定「在哪看」，价格行为决定「怎么做」。**
> 结构给你观察区域（Context），Price Action 给你入场/出场的时机（Timing）。

---

## 一、Price Action 的定位

| 概念 | 看什么 | 回答什么 |
|------|--------|---------|
| Market Structure（宏观） | 趋势、区间、支撑阻力、结构高低点 | 价格"过去做过什么" |
| **Price Action（微观）** | K线力度、情绪、犹豫/强势 | 价格"现在怎么走" |

> ❌ 错误："这根是吞没 → 我做空"
> ✅ 正确：结构确认可以做 → 价格行为确认时机 → 入场

---

## 二、K线的底层逻辑

### 实体（Body）

- **大实体**：一方明显占优 → **突破 / 延续**
- **小实体**：犹豫、拉扯 → **拒绝 / 吸收 / 犹豫**
- 回答：**"谁在控制价格？"**

### 影线（Wick）

- 影线 = 被否定的价格（到过但没站住）
- 长下影线：下方被强力拒绝（买压）
- 长上影线：上方被强力拒绝（卖压）
- 回答：**"哪里被拒绝了？"**

### 核心公式

> 大实体 + 小影线 → **突破 / 延续**
> 小实体 + 长影线 → **拒绝 / 吸收 / 犹豫**
>
> 用实体看"控制力"，用影线看"拒绝力"。

---

## 三、Momentum Candle（动量K线）

### 定义（有明确标准）

- 实体 ≥ 前 3–5 根K线的 **2 倍**
- 理想：3–4 倍
- 影线很短
- 收盘靠近极值

> 代表：**急迫 + 共识 + 单边力量**

### 动量的序列性

- 第一根：启动
- 后续同等大小或更大 → 动量延续
- 明显缩小 → 动量重置

---

## 四、支撑阻力中的 Price Action

### 强支撑的 PA 特征

- 长影线拒绝
- 明确反转形态
- 快速远离（强K线）
- 放量

### 支撑"失效"的信号

- 低点不断降低（压支撑）
- 小K线堆积（犹豫）
- 多次测试
- 放量却不反弹

> **真正的危险不是跌破，而是"撑不起来"**

---

## 五、突破（Breakout）

### 有效突破必须同时满足

1. Momentum Candle
2. 明确收盘越过
3. 放量
4. 关键、明显的水平位

### 突破后两种打法

| 方式 | 说明 |
|------|------|
| 直接追突破 | 需要动量确认 |
| 等回踩确认 | 原阻力 → 支撑 |

### 假突破的典型特征

- 只有影线（没有实体收过）
- 无量
- 收盘失败
- 立刻反向K线

> **假突破 = 最好做反向的机会之一**

---

## 六、Volume 在 PA 中的角色

> Volume 回答的问题：**"有多少人认同这根K线？"**

| 量能 | 含义 |
|------|------|
| 高量 | 共识 |
| 低量 | 偶然 |

### 量价组合

| 组合 | 信号 |
|------|------|
| 高量大K | ✅ 可信 |
| 低量大K | ⚠️ 可疑 |
| 放量反转 | ✅ 可信 |
| 缩量突破 | ⚠️ 警惕 |
| 量价背离 | ⚠️ 趋势衰竭 |

---

## 七、PA 核心心法

- Price Action 不是预测
- 是**在正确的位置，看价格是否给你理由**
- 一切都是概率，不是确定性
- **好交易 = 结构 + 行为 + 风控**

---

## AI 使用指引

- 用户问"这根K线什么意思" → 分析实体/影线 + 所在位置
- 用户问"突破有效吗" → 检查4个有效突破条件
- 用户问"支撑还撑得住吗" → 检查强支撑 vs 失效信号清单
- 永远先确认结构位置，再做 PA 判断
