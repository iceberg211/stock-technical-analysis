# 支撑与阻力 (Support & Resistance)

> **支撑和阻力是"区域"（Zone），不是精确的线！**

---

## 基本概念

| 概念 | 定义 |
|------|------|
| **支撑（Support）** | 价格通常停止下跌的区域，买盘增加 |
| **阻力（Resistance）** | 价格通常停止上涨的区域，卖盘增加 |

---

## 翻转原则（核心规则）

> **支撑被跌破后 → 变成阻力；阻力被突破后 → 变成支撑**

---

## 画法规则

| 规则 | 内容 |
|------|------|
| **规则一** | Zoom Out 看全局，找远观时最明显的水平 |
| **规则二** | 只画最明显的 2–3 个，不要画满屏幕 |
| **规则三** | 覆盖所有触碰点，合并临近水平 |
| **规则四** | 向右延伸，等待未来测试或突破 |

---

## 横盘 vs 趋势市场

**横盘**：支撑买入 / 阻力卖出；关注守住/突破信号

**趋势市场**：关注最近摆动点，可画倾斜趋势线/通道

---

## 突破 vs 守住的判断

### 即将突破的信号
- **更高低点压向阻力**（Higher Lows into Resistance）→ 预期向上突破
- **更低高点压向支撑**（Lower Highs into Support）→ 预期向下突破

### 守住的信号
- 价格快速下探后大幅反弹（强动量 + 快速反弹）
- 历史多次验证（时间越久的历史验证越有效）

---

## 与图表形态的关系

```
双底 = 支撑被两次测试后守住
双顶 = 阻力被两次测试后守住
上升三角形 = 水平阻力 + 上升支撑
下降三角形 = 水平支撑 + 下降阻力
头肩形态 = 颈线就是关键支撑/阻力
```
